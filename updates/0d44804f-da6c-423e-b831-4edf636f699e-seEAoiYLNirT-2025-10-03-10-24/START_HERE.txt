STEP 2: DEPLOY TO RENDER.COM (5 minutes)
═══════════════════════════════════════════════════════════════

1. Go to: https://render.com

2. Sign in or create free account

3. Click blue "New +" button (top right)

4. Select "Static Site"

5. Click "Connect account" to link GitHub

6. Select repository: "cmiecz/GarageVault"

7. Configure exactly like this:
   ┌─────────────────────────────────────────────────────┐
   │ Name: garagevault                                   │
   │ Branch: main                                        │
   │ Root Directory: website                             │
   │ Build Command: [leave empty]                        │
   │ Publish Directory: website                          │
   └─────────────────────────────────────────────────────┘

8. Click "Create Static Site"

9. Wait 2-3 minutes while it deploys ⏳

10. 🎉 COPY YOUR URL! It will look like:
    https://garagevault.onrender.com
    or
    https://garagevault-XXXX.onrender.com

11. Test your pages:
    ✓ https://YOUR-URL.onrender.com/privacy.html
    ✓ https://YOUR-URL.onrender.com/terms.html
